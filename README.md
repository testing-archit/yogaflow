# Yoga Flow 🧘‍♀️✨

A premium, outcome-focused 6-month yoga transformation program grounded in Rishikesh tradition and modern science.

## 🚀 Overview
Yoga Flow is a comprehensive platform built with React and Vite. It offers interactive tools and resources designed to facilitate meditation, detailed Asana guidance, a weekly schedule, and community engagement.

## 🛠️ Tech Stack
- **Frontend**: React (with Vite), TypeScript
- **Backend & Database**: Firebase (Auth, Firestore, Storage)
- **Styling**: Tailwind CSS / Modular Components
- **Payments**: Razorpay Integration

## 📦 Key Features
- **User Dashboard & Weekly Schedule**: Track classes and daily routines.
- **Instructors & Asanas**: Dedicated profiles and deep dive into poses.
- **Meditation & Music**: Embedded tranquil tools for mindfulness.
- **Payment & Admin Dashboard**: Complete flow for subscriptions.

## ⚙️ Running Locally
1. **Clone the repository:**
   ```bash
   git clone https://github.com/testing-archit/yogaflow.git
   cd yogaflow
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Configure Environment Variables:**
   Rename `.env.example` to `.env` and configure your Firebase and Razorpay credentials.
4. **Run the development server:**
   ```bash
   npm run dev
   ```

## 🌐 Deployment
The app integrates out-of-the-box configurations for deployment on platforms like Vercel (see \`vercel.json\`).
